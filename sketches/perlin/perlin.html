Perlin Noise - Flow Field

This is a visualization of a flow field generated with Perlin Noise.
Perlin Noise is a kind of random gradient generator. Random values can be generated in n dimensions with the nice property that, locally, values are similar.
Here, we generate a Perlin Noise grid, and associate each random value \(r\) to a unit vector \([cos(2\pi r), sin(2\pi r)]\), forming essentially a random flow field. We release particles in that flow field and track their position over time.
